#  AI-Based Tech Stack Recommender 

An intelligent recommendation system built using Python that suggests the best tech stack for a project based on user inputs like project type, scale, and requirements.

## 🛠️ Tech Stack
- Python
- scikit-learn
- pandas
- numpy

## ✨ Features
- Takes user input about project requirements
- Recommends suitable tech stack using ML logic
- Simple and interactive interface
- Rule-based + ML hybrid recommendation engine

## 📁 Project Structure
airecommendationsystem/
├── main.py
├── dataset.csv
└── README.md

## 🚀 How to Run

1. Clone the repository
git clone https://github.com/Rashmitha0408/aiRecommendationTask3-RashmithaSai.git

2. Install dependencies
pip install pandas numpy scikit-learn

3. Run the script
python main.py

4. Enter your project details when prompted and get recommendations!

## 📊 How It Works
1. User provides project type and requirements
2. System processes input using trained model
3. Best matching tech stack is recommended
4. Results are displayed with explanation

## 🎯 Use Case
Helps developers and students choose the right technology stack for their projects by analyzing project requirements intelligently.
